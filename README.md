# Blog API

## Description
API REST pour gérer des articles de blog.

## Installation
npm install

## Lancement
node app.js

## Endpoints

- POST /api/articles
- GET /api/articles
- GET /api/articles/{id}
- PUT /api/articles/{id}
- DELETE /api/articles/{id}
- GET /api/articles/search?query=texte

## Documentation
http://localhost:3000/api-docs
